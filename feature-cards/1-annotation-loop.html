Artifact Annotation Loop

Review artifact mà AI vừa viết — ngay trên trình duyệt, click thẳng vào chỗ sai để góp ý.

❌ Vấn đề:Đọc file .md thô trong editor rất khó review. Muốn xem đẹp thì convert tay, rồi lỡ sửa nhầm bản HTML → mất bản gốc, không ai biết ai sửa gì.

✅ AIDLC:Render HTML kiểu Claude tự động, annotate point-and-click, feedback ghi ngược về .md — bản gốc luôn là nguồn sự thật.

CŨ Review thủ công
1. Agent viết xong step.md
2. Đọc markdown thô trong editor — mỏi mắt, khó thấy tổng thể
3. Muốn xem đẹp? Convert tay, mở tool ngoài
4. Lỡ sửa thẳng bản HTML → mất bản gốc .md
5. ⚠ Không ai biết ai sửa gì, lúc nào, vòng mấy
vs
MỚI Annotation loop
1. Agent viết xong step.md
2. Click artifact → popover: Open HTML / Feedback
3. Render HTML kiểu Claude tự động — node, không cần python/pip
4. Bấm A để Annotate → click thẳng vào đoạn cần sửa, thêm note
5. Feedback ghi ngược về .md — file gốc là nguồn sự thật
6. ✓ Áp dụng xong → tự re-render, snapshot lại
.revisions/<artifact>/ rev-1 .md +.html rev-2 .md +.html rev-3 …
Attribution: mỗi vòng gắn tên dev đã sửa (git identity, fallback hostname) + timestamp
Xem lại: selector chọn revision cũ ngay trong HTML render + panel Pipeline History
Skip thông minh: mở lại artifact chưa đổi → không re-render lại
